KRowMapper
====
`KRowMapper`は`Kotlin`向けの`RowMapper`であり、以下のような機能を提供します。

- `BeanPropertyRowMapper`のような、最小限の労力でのオブジェク関係トマッピング（`ORM`）
- リフレクションを用いた関数呼び出しベースの安全なマッピング
- 豊富なデシリアライズ機能による柔軟なマッピング
